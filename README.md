## Program:
#### Configuration:
To configure the enviroment and install the necesary tools just run the <i>requirements.txt</i> with the command below:
```
pip install -r requirements.txt
```
#### Compilation:
Run the command below to run the main.py:
```
python main.py
```
or
```
python3 main.py
```
